# Икар — фича-док (прототип v2)

## Концепт

2D-игра об Икаре, который перемещается **только по воздушным потокам**. Потоки —
видимые «реки» со стрелками и траекторией. Игрок не управляет направлением —
только моментом входа/выхода из потока.

Рефы: течения в Subnautica/Abzû (несут по траектории), Tiny Wings (один инпут),
A Short Hike / Zelda BotW (читаемость апдрафтов).

## Управление

**Одна кнопка — Space (toggle крыльев). Всё.**

| Состояние | В потоке | Вне потока |
|---|---|---|
| Крылья **открыты** | Поток захватывает и несёт вдоль своей траектории | «Парашют»: медленно планирует вниз |
| Крылья **закрыты** | Поток не действует, вылетаешь | Баллистика: инерция + гравитация |

## Петля геймплея

1. Стартуешь в **круговом потоке** — крутишься с открытыми крыльями.
2. Выжидаешь момент → Space (закрыл крылья) → вылет по касательной с инерцией.
3. Летишь дугой (гравитация загибает траекторию вниз).
4. Над следующим потоком → Space (открыл) → если в зоне потока, он подхватывает.
5. Цепочка потоков = уровень. Цель — долететь до солнца.

Скилл = тайминг выхода (где на круге отпустить) и тайминг входа.

## Потоки

- Поток = **полилиния из вэйпоинтов** (трансформы-дети) + ширина захвата + скорость.
- Бывают замкнутые (круг/петля) и открытые (лента). Конец открытого потока
  выбрасывает игрока по касательной.
- Захват: только при открытых крыльях, в пределах ширины. Внутри — скорость
  игрока плавно сводится к «касательная × скорость потока» + притяжение к оси.
- Визуал: стрелки вдоль траектории (направление) + бегущие виспы (скорость/жизнь).

## Фейл и цель

- Внизу море: упал ниже kill-line → мгновенный респавн в стартовый вихрь.
- Цель уровня — солнце. Коснулся → победа (пока: лог + рестарт уровня).

## Принятые дефолты (менять легко, спорить можно)

- Выход из потока: **инерция + гравитация** (дуга), не прямая.
- Крылья вне потока: **парашют** (слабая гравитация, сильный drag) — даёт дотянуть.
- Фейл: рестарт уровня целиком (уровень = один экран).
- Камера: **статичная, весь уровень в кадре** — игрок планирует маршрут как по карте.
- Энергетическая петля «высота↔скорость» — НЕ делаем (решение оператора).

## Скоуп-каты (пока не делаем)

- Ветер-враг, тающий воск, счёт, звук, арт кроме процедурных кружков.
- Редактор потоков — вэйпоинты руками в сцене, этого хватает.

## Уровень 1 (один экран)

```
                                ☀ (12, 8.5)
            B: лента (-5,2)→(9,8), несёт вправо-вверх
   A: круг (центр -9,2, r2.5, CW)
   старт/респавн — верх круга
~~~~~~~~~~~ море (kill, y < -2.5) ~~~~~~~~~~~
```

Маршрут: круг A → выход по касательной вправо с верха → дуга → вход в ленту B →
вынос к солнцу → отпустить в нужный момент.
